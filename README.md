# 📰 News App – Flutter

A Flutter application to read and discover the latest news articles from multiple sources.

## ✨ Features
- 🗞️ Fetch news from public APIs  
- 🔍 Search & filter by categories  
- 📱 Responsive layout  
- 🔔 Push notifications for breaking news  
- 📤 Share articles  

## 🛠 Tech Stack
- Flutter (Dart)
